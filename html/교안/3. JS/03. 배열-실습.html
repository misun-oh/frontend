Q1

배열 기본 조작

🌱 기초

상영작 목록을 배열로 다뤄보세요.

할 일
  • 첫 번째 영화와 마지막 영화를 각각 출력한다 (length 활용)
  • push로 "스텔라 오디세이"를 맨 뒤에 추가한다
  • shift로 맨 앞 영화를 제거하고, 제거된 제목을 출력한다
  • 최종 배열과 length를 출력한다
시작 코드
const nowShowing = ["도시의 밤", "파랑의 계절", "여름의 문"];
✅ 정답 보기
const nowShowing = ["도시의 밤", "파랑의 계절", "여름의 문"];
console.log(nowShowing[0]);
console.log(nowShowing[nowShowing.length - 1]);

nowShowing.push("스텔라 오디세이");
const removed = nowShowing.shift();
console.log(removed);        // 도시의 밤
console.log(nowShowing);     // ["파랑의 계절", "여름의 문", "스텔라 오디세이"]
console.log(nowShowing.length); // 3
Q2

slice vs splice 결과 예측

🌿 응용

원본이 바뀌는지 아닌지에 집중해서 결과를 예측하세요.

주어진 코드/데이터
const a = [10, 20, 30, 40, 50];
const b = a.slice(1, 4);
console.log(b);
console.log(a);

a.splice(1, 3);
console.log(a);
할 일
  • 세 개의 console.log 결과를 예측하고 실행해서 확인한다
💡 힌트
slice의 두 번째 인자는 '끝 인덱스 미포함'입니다.
✅ 정답 보기
// [20, 30, 40]        ← slice(1,4): 인덱스 1~3, 끝(4) 미포함
// [10, 20, 30, 40, 50] ← slice는 원본 유지
// [10, 50]            ← splice(1,3): 인덱스 1부터 3개 제거, 원본 변경
Q3

join으로 장르 문자열 만들기

🌿 응용

카드에 표시되는 "SF · 액션" 같은 문자열을 배열에서 만들어보세요.

할 일
  • join을 사용해 "SF · 액션" 문자열을 만들어 출력한다
  • 요소를 "모험"까지 3개로 늘려도 코드 수정 없이 동작하는지 확인한다
시작 코드
const genres = ["SF", "액션"];
✅ 정답 보기
const genres = ["SF", "액션"];
console.log(genres.join(" · "));   // SF · 액션

genres.push("모험");
console.log(genres.join(" · "));   // SF · 액션 · 모험
Q4

숫자 정렬 함정 고치기

🌳 도전

아래 코드는 의도대로 동작하지 않습니다. 원인을 설명하고 고치세요.

주어진 코드/데이터
const ratings = [9.2, 8.7, 10, 9.8];
ratings.sort();
console.log(ratings); // 기대: [8.7, 9.2, 9.8, 10]  실제: ???
할 일
  • 실제 출력이 무엇인지, 왜 그런지 한 줄로 설명한다
  • 비교 함수를 넣어 오름차순으로 고친다
  • 내림차순 버전도 작성한다
✅ 정답 보기
// 실제: [10, 8.7, 9.2, 9.8] — 문자열 기준이라 "10"이 "8"보다 앞

ratings.sort(function (a, b) { return a - b; }); // 오름차순
ratings.sort(function (a, b) { return b - a; }); // 내림차순
Q5

영화 6편 배열 만들기 + 평점 정렬

🎬 index.html 미션

index.html의 카드 6개(지금은 전부 복사·붙여넣기)를 대신할 진짜 데이터를 main.js에 만듭니다. 아직 문자열 배열 2개로 단순하게 — 5장에서 객체 배열로 업그레이드합니다.

할 일
  • titles 배열에 서로 다른 영화 제목 6개를 담는다
  • ratings 배열에 각 영화의 평점 6개를 같은 순서로 담는다 (그중 1개는 null — 개봉 예정작)
  • for문으로 `제목 - 평점` 형태로 6줄을 출력한다 (null이면 삼항연산자로 "개봉 예정" 표시)
  • [...ratings] 복사본을 내림차순 정렬해서 출력한다 (랭킹 테이블 준비)
💡 힌트
제목과 평점을 배열 2개로 나누니 벌써 불편하죠? 이 불편함이 5장 '객체 배열'을 배우는 이유입니다.
✅ 정답 보기
// main.js (이어서 작성)
const titles  = ["도시의 밤", "파랑의 계절", "스텔라 오디세이",
                 "마지막 항해", "여름의 문", "은하수 특급"];
const ratings = [9.8, 9.5, 9.2, 9.0, 8.7, null];

for (let i = 0; i < titles.length; i++) {
  const label = ratings[i] ? ratings[i] : "개봉 예정";
  console.log(`${titles[i]} - ${label}`);
}

const sorted = [...ratings].sort(function (a, b) { return b - a; });
console.log(sorted); // [9.8, 9.5, 9.2, 9, 8.7, null]
