Q1

let과 const 고르기

🌱 기초

각 상황에 let과 const 중 무엇이 알맞은지 고르고, 실제로 선언해보세요.

할 일
  • ① 영화 제목 (바뀔 일 없음) → ?
  • ② 남은 좌석 수 (예매될 때마다 줄어듦) → ?
  • ③ 원주율 3.14 → ?
  • ④ 현재 페이지 번호 (다음 버튼 누르면 증가) → ?
✅ 정답 보기
const movieTitle = "스텔라 오디세이"; // ① 안 바뀜 → const
let seatsLeft = 30;                   // ② 바뀜 → let
const PI = 3.14;                      // ③ 안 바뀜 → const
let currentPage = 1;                  // ④ 바뀜 → let

Q4

평점 등급 나누기 (if / switch)

🌿 응용

같은 로직을 두 가지 방식으로 작성해보세요.

할 일
  • rating이 9.0 이상 "명작", 7.0 이상 "볼만함", 그 외 "비추"를 출력하는 if~else if~else 작성
  • genre 값("action"/"romance"/"scifi")에 따라 "액션"/"로맨스"/"SF"를 출력하는 switch 작성 (default: "기타")
💡 힌트
switch에서 break를 빼먹으면 아래 case까지 줄줄이 실행됩니다(fall-through).
✅ 정답 보기
const rating = 9.4;
if (rating >= 9.0)      console.log("명작");
else if (rating >= 7.0) console.log("볼만함");
else                    console.log("비추");

const genre = "scifi";
switch (genre) {
  case "action":  console.log("액션");   break;
  case "romance": console.log("로맨스"); break;
  case "scifi":   console.log("SF");     break;
  default:        console.log("기타");
}
Q5

구구단 짝수 단만 출력

🌳 도전

중첩 반복문과 continue를 함께 사용해보세요.

할 일
  • 2단부터 9단까지 출력하되, 홀수 단은 continue로 건너뛴다
  • 각 단은 `2 x 1 = 2` 형태로 1~9까지 출력한다 (템플릿 리터럴 사용)
✅ 정답 보기
for (let dan = 2; dan <= 9; dan++) {
  if (dan % 2 === 1) continue; // 홀수 단 건너뛰기
  for (let i = 1; i <= 9; i++) {
    console.log(`${dan} x ${i} = ${dan * i}`);
  }
}
Q6

영화 1편 변수화 + 삼항연산자 배지

🎬 index.html 미션

index.html 카드에 하드코딩된 영화 정보를 main.js의 변수로 옮기고, 평점이 있으면 배지 문자열, 없으면 빈 문자열을 만드는 삼항연산자를 작성합니다. (아직 화면은 안 바꿉니다 — 콘솔로만!)

할 일
  • main.js에 movieTitle(제목), movieGenre(장르), movieRating(평점 9.2) 변수를 const로 선언한다
  • 평점 등급(명작/볼만함/비추)을 if문으로 콘솔에 출력한다
  • const ratingBadge = movieRating ? `⭐ ${movieRating}` : ""; 를 작성한다
  • movieRating을 null로 바꿔보고 ratingBadge가 빈 문자열이 되는 것도 확인한다
💡 힌트
null은 조건식에서 false로 취급됩니다(거짓 같은 값). 그래서 rating ? A : B 패턴이 동작합니다.
✅ 정답 보기
// main.js (이어서 작성)
const movieTitle = "스텔라 오디세이";
const movieGenre = "SF · 액션";
const movieRating = 9.2;   // null로 바꿔서도 테스트!

if (movieRating >= 9.0)      console.log("명작");
else if (movieRating >= 7.0) console.log("볼만함");
else                         console.log("비추");

const ratingBadge = movieRating ? `⭐ ${movieRating}` : "";
console.log(`[${movieTitle}] ${ratingBadge}`);
// movieRating = 9.2  → [스텔라 오디세이] ⭐ 9.2
// movieRating = null → [스텔라 오디세이]
