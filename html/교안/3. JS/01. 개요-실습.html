Q2

개발자 도구에서 첫 출력

🌱 기초

브라우저에서 F12를 눌러 Console 패널을 열고, 직접 입력해보세요.

할 일
  • console.log("안녕하세요") 를 입력하고 결과를 확인한다
  • console.log(10 + 20) 을 입력하고 결과를 확인한다
  • 10 + 20 만 입력했을 때와의 차이(반환값 표시)를 관찰한다
✅ 정답 보기
console.log("안녕하세요"); // 안녕하세요
console.log(10 + 20);      // 30
// 콘솔에 식만 입력하면 평가된 반환값이 회색으로 표시됩니다.
Q3

script 위치 실험

🌿 응용

script 위치에 따라 실행 시점이 달라지는 것을 직접 확인하세요.

주어진 코드/데이터
<!DOCTYPE html>
<html>
<head>
  <script>
    console.log(document.querySelector("h1")); // 여기서는?
  </script>
</head>
<body>
  <h1>무비 컬렉션</h1>
  <script>
    console.log(document.querySelector("h1")); // 여기서는?
  </script>
</body>
</html>
할 일
  • 위 파일을 만들어 브라우저로 열고 콘솔을 확인한다
  • head의 출력과 body 끝의 출력이 왜 다른지 한 줄로 설명한다
💡 힌트
브라우저는 HTML을 위에서 아래로 한 줄씩 읽습니다.
✅ 정답 보기
// head 쪽 출력  : null      (h1이 아직 만들어지기 전에 실행됨)
// body 끝 출력 : <h1>…</h1> (h1이 만들어진 후 실행됨)
// → 브라우저는 HTML을 위에서 아래로 읽기 때문에,
//   요소를 다루는 스크립트는 </body> 직전에 두는 것이 기본!
