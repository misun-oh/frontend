MOVIE COLLECTION JS · CHAPTER 01

01. 개요

웹의 구조 속에서 자바스크립트가 하는 일, 그리고 코드를 작성·실행하는 방법

🎯 학습 목표

  • ☐ 클라이언트-서버 구조와 요청/응답 흐름을 설명할 수 있다
  • ☐ HTML·CSS·자바스크립트의 역할 차이를 설명할 수 있다
  • ☐ script 태그로 자바스크립트를 작성하고 개발자 도구로 확인할 수 있다
  • ☐ 외부 .js 파일을 만들어 HTML에 연결할 수 있다
01-1

클라이언트-서버 구조

웹은 클라이언트(브라우저)가 요청(Request)을 보내고 서버가 응답(Response)을 돌려주는 구조로 동작합니다. 이때 서로 약속된 통신 규칙을 프로토콜이라고 합니다.

프로토콜역할
HTTP하이퍼텍스트 전송 규약 — 웹 페이지를 주고받는 기본 통신
FTP대량의 파일을 전송/수신하기 위한 파일 전송 규약
TELNET원격지에서 서버 컴퓨터를 원격 제어하기 위한 규약

우리가 10장에서 배울 fetch도 결국 이 HTTP 요청/응답을 자바스크립트로 직접 보내는 기능입니다.

클라이언트 언어역할

01-3

자바스크립트란

자바스크립트(JavaScript)는 웹 브라우저에서 가장 많이 사용하는 인터프리터 방식의 객체지향 프로그래밍 언어로, ECMA(ECMAScript) 표준을 따르는 대표적인 웹 기술입니다.

용어의미
인터프리터 방식코드를 한 줄씩 읽으며 바로 실행 (별도 컴파일 과정 없음)
객체지향데이터와 기능을 '객체'라는 단위로 묶어 다루는 방식 (5장에서 학습)
ECMAScript자바스크립트의 표준 명세. ES6(2015)부터 let/const 등 최신 문법 도입
⚠️ 자바(Java)와 자바스크립트(JavaScript)는 전혀 다른 언어

이름만 비슷할 뿐 문법·용도·실행 환경이 모두 다릅니다. 마케팅 목적으로 비슷한 이름이 붙었을 뿐입니다.

01-4

개발자 도구와 console.log

브라우저마다 개발자 도구(Developer Tools)가 있으며, 크롬에서는 F12 키(또는 우클릭 → 검사)로 엽니다.

console.log()는 브라우저 화면이 아니라 개발자 도구의 Console 패널에 값을 출력하며, 스크립트를 디버깅할 때 가장 자주 사용합니다.

console.log("무비 컬렉션 시작!");
console.log(1 + 2);
실행 결과
무비 컬렉션 시작!
3
⭐ 이 과정의 모든 확인은 console.log부터

1~6장까지는 결과를 대부분 콘솔로 확인하고, 8장(DOM)부터 화면에 그리기 시작합니다. F12 → Console을 여는 습관을 지금 들이세요.

01-5

script 태그 — 작성 위치와 방법

자바스크립트는 HTML의 <script></script> 태그 안에 작성합니다. type 속성은 기본값이라 생략 가능합니다.

<script>
  // 여기에 자바스크립트 코드 작성
</script>

<script>는 <head>와 <body> 어디에나 작성할 수 있지만, 위치에 따라 실행 시점이 달라집니다.

⚠️ script 위치가 중요한 이유

브라우저는 HTML을 위에서 아래로 읽습니다. <head>에서 스크립트가 아직 만들어지지 않은 태그를 선택하려 하면 null이 나옵니다. 그래서 실무에서는 </body> 닫히기 직전에 script를 두는 것이 기본입니다. (우리 index.html도 이 방식)

과거에 쓰이던 language 속성은 폐기되었고, charset 속성은 meta 태그가 대신하므로 쓸 필요가 없습니다.

01-6

외부 .js 파일 분리와 연결

코드가 길어지면 HTML 안에 두지 않고 별도의 .js 파일로 분리한 뒤 src 속성으로 연결합니다. 유지보수와 재사용에 유리하고, 이 과정에서도 main.js 파일 하나에 기능을 계속 쌓아갈 것입니다.

<!-- index.html : </body> 닫히기 직전 -->
<script src="main.js"></script>
⚠️ src를 쓰면 태그 안의 코드는 무시됩니다

<script src="main.js">console.log(1)</script>처럼 써도 안쪽 코드는 실행되지 않습니다. 외부 파일 연결용 태그와 직접 작성용 태그는 분리하세요.

🔗 index.html 연결 포인트

실습 마지막 미션에서 main.js를 만들어 index.html에 연결합니다. 이 파일이 11장까지 계속 자라나 최종 프로젝트가 됩니다.
