06. 내장객체

자바스크립트가 기본으로 제공하는 객체들 — Object · Number · String · Date · Math

🎯 학습 목표

  • ☐ 모든 객체가 Object를 상속한다는 구조를 이해한다
  • ☐ Number의 toFixed 등 숫자 표시 메서드를 쓸 수 있다
  • ☐ Date 객체로 날짜를 만들고 년/월/일을 꺼낼 수 있다

06-1

Object 객체

모든 기본 내장 객체는 Object 객체를 상속합니다. 즉 모든 객체는 Object의 메서드를 기본으로 가지고 있습니다.

메서드설명
hasOwnProperty(name)객체가 name 속성을 (상속 말고) 직접 갖고 있는지 확인
toString()객체를 문자열로 변환
toLocaleString()지역/언어 설정에 맞는 문자열로 변환 — 예: 숫자 천 단위 콤마
valueOf()객체의 원시 값을 나타냄
constructor객체를 만든 생성자 함수를 가리키는 속성
const movie = { title: "도시의 밤" };
console.log(movie.hasOwnProperty('title'));  // true
console.log((1234567).toLocaleString());     // "1,234,567"
06-2

Number 객체

메서드설명
toFixed(자릿수)고정 소수점 표기 — 문자열을 리턴!
toExponential()지수 표기 문자열
toPrecision(n)길이에 따라 지수/고정 소수점 표기 문자열
속성설명
Number.MAX_VALUE / MIN_VALUE표현 가능한 최대/최소(0에 가장 가까운 양수) 숫자
Number.NaN숫자로 나타낼 수 없는 값
Number.POSITIVE_INFINITY / NEGATIVE_INFINITY양/음의 무한대
const avg = (9.8 + 9.5 + 9.2) / 3;
console.log(avg);              // 9.499999999999998
console.log(avg.toFixed(1));   // "9.5"
⚠️ toFixed는 문자열을 리턴

avg.toFixed(1) + 1은 "9.51"이 됩니다(문자열 연결). 계산을 이어가려면 Number(avg.toFixed(1))로 되돌리세요.

06-3

String 객체의 HTML 관련 메서드 (레거시)

String 객체에는 문자열을 태그로 감싸 리턴하는 메서드들이 있습니다: bold()(<b>), italics()(<i>), link(url)(<a href>), fontcolor(색), big(), small(), sub(), sup() 등.

⚠️ 전부 deprecated(폐기 예정) — 시험·레거시 코드용 지식

이 메서드들이 만드는 <font>, <big>, <blink> 태그 자체가 HTML 표준에서 퇴출되었습니다. 실무에서는 템플릿 리터럴로 직접 태그를 조립합니다(4장 makeCard처럼). '옛 교재에 나오는 이유'만 알아두세요.

06-4

Date 객체

날짜를 관리하는 객체로, new 키워드로 생성합니다.

const now = new Date();                    // 현재 날짜·시간
const day1 = new Date(2026, 6, 13);        // 2026년 7월 13일 (월은 0부터!)
const day2 = new Date('2026-07-13');       // 문자열로도 생성 가능
메서드 (get/set 쌍)설명 · 범위
getFullYear()연도 (예: 2026)
getMonth()월 — 0(1월) ~ 11(12월) ⚠️
getDate()일 (1 ~ 31)
getDay()요일 — 0(일요일) ~ 6(토요일) · setDay는 없음
getHours() / getMinutes() / getSeconds()시(0~23) / 분(0~59) / 초(0~59)
getTime()1970-01-01 기준 경과 밀리초 — 날짜 간 차이 계산에 사용
⚠️ getMonth()는 0부터

7월이면 6이 나옵니다. 화면에 표시할 때는 getMonth() + 1을 잊지 마세요. 신입 개발자 단골 버그 1위입니다.

const now = new Date();
const dateText = `${now.getFullYear()}.${now.getMonth() + 1}.${now.getDate()} 기준`;
console.log(dateText);   // 예: 2026.7.13 기준
06-5

Math 객체 활용 — 정수 난수 공식

2장에서 본 Math 메서드 중 Math.random()(0 이상 1 미만 소수)과 Math.floor()(내림)를 조합하면 범위 내 정수 난수를 만들 수 있습니다.

// 0 ~ (n-1) 정수: Math.floor(Math.random() * n)
const idx = Math.floor(Math.random() * 6);   // 0~5 — 배열 인덱스로 딱!

// min ~ max 정수: Math.floor(Math.random() * (max - min + 1)) + min
const dice = Math.floor(Math.random() * 6) + 1; // 1~6
