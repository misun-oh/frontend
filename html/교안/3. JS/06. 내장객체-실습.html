Q1

평균 평점 소수 첫째 자리

🌱 기초

평점 평균을 보기 좋게 표시하세요.

할 일
  • 세 평점의 평균을 구한다
  • toFixed(1)로 소수 첫째 자리까지 표시한다
  • toFixed의 결과에 +1을 해보고 왜 이상한 결과가 나오는지 설명한다
시작 코드
const ratings = [9.8, 9.5, 9.2];
✅ 정답 보기
const ratings = [9.8, 9.5, 9.2];
const avg = (ratings[0] + ratings[1] + ratings[2]) / 3;
console.log(avg.toFixed(1));       // "9.5"
console.log(avg.toFixed(1) + 1);   // "9.51" — toFixed는 문자열이라 연결됨!
console.log(Number(avg.toFixed(1)) + 1); // 10.5 — 숫자로 되돌린 뒤 계산
Q2

오늘 날짜 정보 출력

🌱 기초

Date 객체에서 정보를 꺼내보세요.

할 일
  • new Date()로 현재 날짜 객체를 만든다
  • 연 / 월 / 일 / 요일 숫자를 각각 출력한다 (월은 +1 보정)
  • 요일 숫자를 ["일","월","화","수","목","금","토"] 배열로 한글 요일로 바꿔 출력한다
💡 힌트
getDay()는 0(일요일)~6(토요일)을 리턴하므로 배열 인덱스로 쓰기 딱 좋습니다.
✅ 정답 보기
const now = new Date();
console.log(now.getFullYear());
console.log(now.getMonth() + 1);   // 0부터 시작하므로 +1
console.log(now.getDate());

const dayNames = ["일", "월", "화", "수", "목", "금", "토"];
console.log(dayNames[now.getDay()] + "요일");
Q3

주사위 함수 만들기

🌿 응용

정수 난수 공식을 함수로 만들어보세요.

할 일
  • function rollDice() — 1~6 정수를 리턴하는 함수를 만든다
  • for문으로 10번 굴려 결과를 출력한다
  • function randomBetween(min, max) 로 일반화한다
✅ 정답 보기
function rollDice() {
  return Math.floor(Math.random() * 6) + 1;
}
for (let i = 0; i < 10; i++) console.log(rollDice());

function randomBetween(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min;
}
console.log(randomBetween(1, 45)); // 로또 번호처럼 1~45
Q4

디데이 계산기

🌳 도전

getTime()으로 두 날짜의 차이를 계산하세요.

할 일
  • openDay = new Date("2026-12-25") 개봉일을 만든다
  • getTime() 차이(밀리초)를 구한다
  • 밀리초를 일 단위로 환산해 "개봉까지 D-n" 형태로 출력한다 (1일 = 1000*60*60*24, Math.ceil 사용)
💡 힌트
밀리초 → 초(÷1000) → 분(÷60) → 시(÷60) → 일(÷24) 순으로 나눕니다.
✅ 정답 보기
const openDay = new Date("2026-12-25");
const now = new Date();

const diffMs = openDay.getTime() - now.getTime();
const diffDays = Math.ceil(diffMs / (1000 * 60 * 60 * 24));
console.log(`개봉까지 D-${diffDays}`);
Q5

오늘의 추천 영화 + 갱신일

🎬 index.html 미션

5장에서 만든 movies 객체 배열을 활용해, 11장에서 화면에 붙일 두 가지 재료를 main.js에 만듭니다.

할 일
  • function pickTodayMovie() — movies에서 무작위 1편을 리턴하는 함수를 만든다 (Math.random + Math.floor + length)
  • 리턴된 영화의 제목과 평점 배지(삼항연산자)를 콘솔에 출력한다
  • function getUpdateText() — "2026.7.13 기준" 형태의 갱신일 문자열을 리턴하는 함수를 만든다 (월 +1 보정!)
  • 두 함수를 각각 호출해 결과를 확인한다
✅ 정답 보기
// main.js (이어서 작성)
function pickTodayMovie() {
  const idx = Math.floor(Math.random() * movies.length);
  return movies[idx];
}

const today = pickTodayMovie();
const badge = today.rating ? `⭐ ${today.rating}` : "개봉 예정";
console.log(`오늘의 추천: ${today.title} ${badge}`);

function getUpdateText() {
  const now = new Date();
  return `${now.getFullYear()}.${now.getMonth() + 1}.${now.getDate()} 기준`;
}
console.log(getUpdateText()); // 예: 2026.7.13 기준
