Q4

랭킹 테이블 tbody 다시 그리기

🌳 도전

3장의 sort와 innerHTML을 조합해 랭킹 테이블을 데이터로 그려보세요.

주어진 코드/데이터
const ranking = [
  { title: "여름의 문",   rating: 8.7 },
  { title: "도시의 밤",   rating: 9.8 },
  { title: "파랑의 계절", rating: 9.5 },
];
할 일
  • [...ranking]을 평점 내림차순으로 정렬한다
  • forEach로 순위제목평점 문자열을 누적한다 (순위 = 인덱스 + 1)
  • tbody 요소를 선택해 innerHTML에 대입한다
💡 힌트
forEach 콜백의 두 번째 매개변수가 인덱스입니다 — 순위는 i + 1.
✅ 정답 보기
const sorted = [...ranking].sort(function (a, b) {
  return b.rating - a.rating;
});

let rows = "";
sorted.forEach(function (m, i) {
  rows += `<tr><td>${i + 1}</td><td>${m.title}</td><td>${m.rating}</td></tr>`;
});

document.querySelector("tbody").innerHTML = rows;
Q5

카드 6장 동적 렌더링 (핵심 전환점!)

🎬 index.html 미션

index.html에 복사·붙여넣기된 카드 6개를 전부 삭제하고, 5장의 movies 배열 + makeCard()로 화면을 데이터로 그립니다. 평점 배지가 드디어 눈에 보입니다.

주어진 코드/데이터
/* styles.css에 추가 — 배지 위치 (카드 오른쪽 상단) */
.card { position: relative; }
.rating-badge {
  position: absolute; top: 8px; right: 8px;
  background: rgba(0,0,0,.75); color: gold;
  padding: 4px 10px; border-radius: 999px;
  font-size: 13px; font-weight: 700;
}
할 일
  • index.html의
    …
    6개를 전부 삭제한다 (grid div는 남긴다!)
  • 위 CSS를 styles.css에 추가한다
  • main.js에 function renderMovies(list) 를 만든다 — forEach + makeCard로 html 문자열을 누적해 .grid의 innerHTML에 대입
  • renderMovies(movies) 를 호출해 카드 6장이 그려지는지 확인한다
  • rating이 null인 영화만 배지가 안 뜨는지 확인한다 (2장부터 이어온 삼항연산자의 완성!)
  • 도전: 8-4번 문제를 함수화해 renderRanking(movies)도 만들어 실행한다
💡 힌트
renderMovies를 '함수'로 만드는 이유: 9장에서 검색 결과 배열로 renderMovies(filtered)를 다시 부르기만 하면 필터링이 완성되기 때문입니다.
✅ 정답 보기
// main.js (이어서 작성)
function renderMovies(list) {
  const grid = document.querySelector(".grid");
  let html = "";
  list.forEach(function (movie) {
    html += makeCard(movie);
  });
  grid.innerHTML = html;
}

function renderRanking(list) {
  const sorted = [...list]
    .filter(function (m) { return m.rating; })  // 평점 없는 작품 제외
    .sort(function (a, b) { return b.rating - a.rating; });
  let rows = "";
  sorted.forEach(function (m, i) {
    rows += `<tr><td>${i + 1}</td><td>${m.title}</td><td>${m.rating}</td></tr>`;
  });
  document.querySelector("tbody").innerHTML = rows;
}

renderMovies(movies);
renderRanking(movies);
