03. 배열

여러 값을 순서대로 담는 그릇 — 영화 6편을 데이터로 다루는 첫 걸음

🎯 학습 목표

  • ☐ sort()에 콜백 함수를 넣어 원하는 기준으로 정렬할 수 있다

03-1

배열이란 / 선언과 초기화

배열은 다양한 타입의 데이터를 순서대로 보관하는 변수 모음입니다. 자료형 제한이 없어 숫자·문자열·불리언·객체·함수까지 모두 담을 수 있습니다.

// [ ] 리터럴 방식 — 생성과 초기화를 동시에 (권장)
const titles = ["도시의 밤", "파랑의 계절", "스텔라 오디세이"];

// new Array() 방식
const arr1 = new Array();        // 빈 배열
const arr2 = new Array(5);       // 길이 5짜리 빈 배열
const arr3 = new Array(1, 2, 3); // [1, 2, 3]
⚠️ new Array(숫자 하나)의 함정

new Array(5)는 [5]가 아니라 길이가 5인 빈 배열을 만듭니다. 헷갈림을 피하려면 항상 [ ] 리터럴을 쓰세요.

03-2

인덱스로 값 읽고 쓰기

배열의 각 칸에는 0부터 시작하는 번호(인덱스)가 붙습니다. 번호 범위는 0 ~ (길이-1)입니다.

const titles = ["도시의 밤", "파랑의 계절", "스텔라 오디세이"];

console.log(titles[0]);        // 첫 번째 값 읽기
console.log(titles.length);    // 배열 길이
console.log(titles[titles.length - 1]); // 마지막 값

titles[1] = "여름의 문";       // 값 바꾸기
실행 결과
도시의 밤
3
스텔라 오디세이

⭐ slice vs splice — 이름은 비슷, 동작은 정반대

slice는 원본을 그대로 두고 복사본을 리턴하고, splice는 원본을 직접 수술합니다. 원본 변경 여부(O/X) 열을 꼭 기억하세요.

const list = ["a", "b", "c", "d"];
console.log(list.slice(1, 3));  // ["b", "c"]  (원본 유지)
console.log(list);              // ["a", "b", "c", "d"]

list.splice(1, 2);              // 인덱스 1부터 2개 제거 (원본 변경!)
console.log(list);              // ["a", "d"]
03-4

sort() — 콜백으로 정렬 기준 바꾸기

sort()는 기본적으로 문자열 기준 오름차순으로 정렬합니다. 그래서 숫자를 그냥 정렬하면 이상한 결과가 나옵니다.

const nums = [30, 4, 100];
console.log(nums.sort());           // [100, 30, 4] ?! (문자열 "1","3","4" 순)

정렬 방법을 바꾸려면 sort()의 인자에 비교 함수를 넣습니다. 두 매개변수(left, right)를 받아, 음수를 리턴하면 left가 앞으로 갑니다.

const ratings = [9.2, 8.7, 9.8, 9.0];

ratings.sort(function (left, right) {
  return left - right;   // 오름차순
});

ratings.sort(function (left, right) {
  return right - left;   // 내림차순
});
console.log(ratings);
실행 결과
[9.8, 9.2, 9, 8.7]
⚠️ sort는 원본을 바꿉니다

원본을 지키고 싶으면 const sorted = [...arr].sort(...)처럼 복사본을 만들어 정렬하세요. (스프레드 ...은 5장에서 자세히)

03-5

배열 순회 — for · for...in · forEach

const titles = ["도시의 밤", "파랑의 계절", "스텔라 오디세이"];

// ① 기본 for문 — 인덱스가 필요할 때
for (let i = 0; i < titles.length; i++) {
  console.log(i + "위 : " + titles[i]);
}

// ② for...in — 인덱스(키)를 순회
for (const i in titles) {
  console.log(titles[i]);
}

// ③ forEach — 요소를 하나씩 콜백으로 전달 (실무에서 가장 많이)
titles.forEach(function (title, i) {
  console.log(i, title);
});
⭐ 배열 순회는 forEach(또는 기본 for)가 기본

for...in은 원래 객체용이라(5장) 배열에서는 예상 밖 동작이 생길 수 있습니다. 배열에는 forEach / for / for...of를 쓰는 게 안전합니다.
