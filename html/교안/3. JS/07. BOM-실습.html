Q4

localStorage л°©л¬ё нҡҹмҲҳ м№ҙмҡҙн„°

рҹҢі лҸ„м „

мғҲлЎңкі м№Ён•ҙлҸ„ лӮЁлҠ” лҚ°мқҙн„°лҘј л§Ңл“Өм–ҙліҙм„ёмҡ”.

н•  мқј
  • localStorageм—җм„ң "visit" к°’мқ„ мқҪлҠ”лӢӨ (мІҳмқҢмқҙл©ҙ null)
  • nullмқҙл©ҙ 0мңјлЎң мӢңмһ‘, м•„лӢҲл©ҙ Number()лЎң мҲ«мһҗ ліҖнҷҳн•ңлӢӨ (м ҖмһҘмҶҢ к°’мқҖ л¬ёмһҗм—ҙ!)
  • 1 мҰқк°ҖмӢңмјң лӢӨмӢң м ҖмһҘн•ҳкі , `nлІҲм§ё л°©л¬ёмһ…лӢҲлӢӨ` лҘј м¶ңл Ҙн•ңлӢӨ
  • мғҲлЎңкі м№Ёмқ„ м—¬лҹ¬ лІҲ н•ҳл©° мҲ«мһҗк°Җ лҠҳм–ҙлӮҳлҠ”м§Җ нҷ•мқён•ңлӢӨ
рҹ’Ў нһҢнҠё
getItemмқҖ н•ӯмғҒ л¬ёмһҗм—ҙ(лҳҗлҠ” null)мқ„ лҰ¬н„ҙн•©лӢҲлӢӨ. 2мһҘмқҳ нҳ•ліҖнҷҳмқҙ м—¬кё°м„ң м“°мһ…лӢҲлӢӨ.
вң… м •лӢө ліҙкё°
let visit = localStorage.getItem("visit");
visit = visit ? Number(visit) : 0;   // л¬ёмһҗм—ҙ вҶ’ мҲ«мһҗ ліҖнҷҳ н•„мҲҳ!

visit++;
localStorage.setItem("visit", visit);
console.log(`${visit}лІҲм§ё л°©л¬ёмһ…лӢҲлӢӨ`);
Q5

н—ӨлҚ”м—җ мӢӨмӢңк°„ мӢңкі„ лӢ¬кё°

рҹҺ¬ index.html лҜём…ҳ

л“ңл””м–ҙ нҷ”л©ҙмқҙ мІҳмқҢмңјлЎң мӣҖм§Ғмһ…лӢҲлӢӨ! setInterval + DateлЎң н—ӨлҚ”м—җ нҳ„мһ¬ мӢңк°Ғмқ„ н‘ңмӢңн•©лӢҲлӢӨ. (мҡ”мҶҢ м„ нғқВ·н‘ңмӢң мҪ”л“ңлҠ” 8мһҘ лӮҙмҡ©мқҙлқј м•„лһҳм—җ м ңкіө вҖ” лӢӨмқҢ мһҘм—җм„ң л°°мӣҒлӢҲлӢӨ)

мЈјм–ҙм§„ мҪ”л“ң/лҚ°мқҙн„°
<!-- index.html н—ӨлҚ”мқҳ лЎңкі  мҳҶм—җ м¶”к°Җ -->
<div class="logo">рҹҺ¬ л¬ҙл№„ м»¬л үм…ҳ <span id="clock"></span></div>
н•  мқј
  • index.html лЎңкі  spanм—җ id="clock"мқё л№Ҳ spanмқ„ м¶”к°Җн•ңлӢӨ (мң„ м°ёкі )
  • main.jsм—җ function showClock() мқ„ л§Ңл“ лӢӨ вҖ” new Date()м—җм„ң мӢң:л¶„:мҙҲлҘј кәјлӮҙ л¬ёмһҗм—ҙлЎң мЎ°лҰҪ
  • м ңкіө мҪ”л“ң document.querySelector("#clock").textContent = мӢңк°Ғл¬ёмһҗм—ҙ; лЎң нҷ”л©ҙм—җ н‘ңмӢңн•ңлӢӨ
  • setInterval(showClock, 1000) мңјлЎң 1мҙҲл§ҲлӢӨ к°ұмӢ н•ҳкі , showClock()мқ„ н•ң лІҲ мҰүмӢң нҳём¶ңн•ҙ мІҳмқҢ 1мҙҲ кіөл°ұмқ„ м—Ҷм•ӨлӢӨ
рҹ’Ў нһҢнҠё
нҷ”л©ҙм—җ 8:5:3мІҳлҹј лӮҳмҳЁлӢӨл©ҙ? лҸ„м „кіјм ң: н•ң мһҗлҰ¬ мҲ«мһҗ м•һм—җ 0мқ„ л¶ҷм—¬ 08:05:03мңјлЎң л§Ңл“Өм–ҙліҙм„ёмҡ” (мӮјн•ӯм—°мӮ°мһҗ: h < 10 ? "0"+h : h).
вң… м •лӢө ліҙкё°
// main.js (мқҙм–ҙм„ң мһ‘м„ұ)
function showClock() {
  const now = new Date();
  const h = now.getHours();
  const m = now.getMinutes();
  const s = now.getSeconds();
  const text = `${h}:${m}:${s}`;
  document.querySelector("#clock").textContent = text; // вҶҗ 8мһҘм—җм„ң л°°мҡё н•ң мӨ„!
}

showClock();                  // мҰүмӢң 1нҡҢ мӢӨн–ү
setInterval(showClock, 1000); // мқҙнӣ„ 1мҙҲл§ҲлӢӨ к°ұмӢ
