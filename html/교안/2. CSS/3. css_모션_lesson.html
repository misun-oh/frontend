화면 속 움직임을 설계하는 두 가지 도구,
transition과 transform

속도(duration)와 모양 변화(transform)를 함께 이해하면, 정지된 화면이 살아있는 인터페이스로 바뀝니다. 아래에서 값을 직접 바꿔보며 눈으로 확인해보세요.

transition-duration: 2.6s · transform: translate + rotate + scale
FRAME 01

transition-duration

상태 변화에 걸리는 "시간"을 정하는 속성

transition은 어떤 CSS 속성값이 바뀔 때(예: 색이 바뀌거나 크기가 커질 때), 그 변화를 순간적으로 처리하지 않고 부드럽게 이어지도록 만드는 속성입니다. 그중 transition-duration은 그 변화가 몇 초에 걸쳐 일어날지를 정합니다.

/* 기본 문법 */ .box { transition-property: background-color; transition-duration: 0.4s; transition-timing-function: ease; transition-delay: 0s; } /* 한 줄로 줄여쓰기 (shorthand) */ .box { transition: background-color 0.4s ease; }

transition 관련 4가지 속성

속성역할예시
transition-property어떤 속성을 애니메이션 할지 지정all, background-color, transform
transition-duration변화에 걸리는 시간0.3s, 500ms
transition-timing-function속도의 완급 곡선ease, linear, ease-in-out
transition-delay변화가 시작되기까지 대기 시간0.2s

transition-timing-function 종류

값 설명 움직임 느낌
linear 처음부터 끝까지 일정한 속도 균일하게 쭉
ease 기본값. 천천히 시작 → 빨라짐 → 천천히 끝남 가장 자연스러움
ease-in 천천히 시작해서 점점 빨라짐 끝이 갑작스러움
ease-out 빠르게 시작해서 점점 느려짐 끝이 부드러움
ease-in-out 시작도 끝도 부드러움 양쪽 다 완만함
cubic-bezier(x1,y1,x2,y2) 4개 숫자로 곡선을 직접 지정 (커스텀) 튕기는 효과 등 자유 설정
steps(n) n개의 구간으로 뚝뚝 끊어서 전환 계단식 (스프라이트 애니메이션)

transform

    
요소의 위치·각도·크기를 바꾸는 "모양 변화" 속성


    
transform은 요소를 이동시키거나(translate), 회전시키거나(rotate), 크기를 바꾸거나(scale), 기울이는(skew) 데 사용합니다. transform 자체는 순간적으로 적용되지만, transition과 함께 쓰면 그 변화가 부드럽게 보이게 됩니다.


    
/* 자주 쓰는 transform 함수 */
.box {
  transform: translateX(40px);   /* 이동 */
  transform: rotate(45deg);      /* 회전 */
  transform: scale(1.5);        /* 확대/축소 */
  transform: skewX(15deg);      /* 기울이기 */

  /* 여러 개를 동시에, 공백으로 연결 */
  transform: translateX(40px) rotate(45deg) scale(1.2);
}


    

      
함수변화예시

      
translate(x, y)위치 이동translateX(50px)

      
rotate(deg)회전rotate(180deg)

      
scale(n)크기 확대/축소scale(0.5)

      
skew(x, y)기울이기skewX(20deg)

transition + transform 함께 쓰기

    
실제 UI에서는 두 속성을 같이 사용하는 경우가 대부분입니다. transform으로 "어떻게 변할지"를 정하고, transition으로 "얼마나 부드럽게 변할지"를 정합니다.

    

      
카드에 마우스를 올려보세요

      

        

          hover card
        

      

      
.card {
  transition: transform 0.25s ease, box-shadow 0.25s ease;
}
.card:hover {
  transform: translateY(-8px) scale(1.04);
  box-shadow: 0 14px 24px rgba(0,0,0,0.14);
}

연습문제

    
코드를 먼저 작성해보고, 막히면 정답을 펼쳐보세요


    

      EXERCISE 1
      
박스에 마우스를 올리면 0.5초에 걸쳐 배경색이 주황색(#FF5A36)으로 바뀌도록 만들어보세요.

      

        정답 보기
        
.box {
  background: #2F49E8;
  transition: background-color 0.5s ease;
}
.box:hover {
  background-color: #FF5A36;
}

      

    


    

      EXERCISE 2
      
박스에 마우스를 올리면 오른쪽으로 100px 이동하면서 동시에 45도 회전하도록 만들어보세요. 변화는 0.3초 동안 일어나야 합니다.

      

        정답 보기
        
.box {
  transition: transform 0.3s ease;
}
.box:hover {
  transform: translateX(100px) rotate(45deg);
}

      

    


    

      EXERCISE 3 · 응용
      
버튼에 마우스를 올리면 1.1배 커지고, 위로 4px 떠오르고, 그림자가 진해지는 효과를 0.2초 동안 만들어보세요. (힌트: transform과 box-shadow를 동시에 transition 하려면 쉼표로 구분합니다)

      

        정답 보기
        
.button {
  transition: transform 0.2s ease, box-shadow 0.2s ease;
}
.button:hover {
  transform: translateY(-4px) scale(1.1);
  box-shadow: 0 10px 20px rgba(0,0,0,0.2);
}
