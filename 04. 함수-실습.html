Q2

화살표 함수로 바꿔쓰기

🌱 기초

교안 04-3의 생략 규칙을 적용해, 아래 함수들을 화살표 함수로 바꿔보세요.

주어진 코드/데이터
function double(n) { return n * 2; }

function makeBadge(rating) {
  const symbol = rating >= 9 ? "🏆" : "⭐";
  return `${symbol} ${rating}`;
}
할 일
  • double을 매개변수 괄호 생략 + 중괄호/return 생략 형태의 화살표 함수로 바꾼다
  • makeBadge는 본문이 여러 줄이므로 중괄호와 return을 그대로 유지한 화살표 함수로 바꾼다
  • 두 함수를 각각 호출해서 원본과 결과가 같은지 확인한다
💡 힌트
본문이 식 하나뿐이면 중괄호와 return을 함께 지워야 합니다 — 중괄호만 남기고 return을 빼먹으면 undefined가 나옵니다(04-3 실수 사례 참고).
✅ 정답 보기
const double = n => n * 2;              // 매개변수 1개 + 식 1개 → 최대 축약
console.log(double(5));                 // 10

const makeBadge = (rating) => {          // 여러 줄이라 중괄호+return 필요
  const symbol = rating >= 9 ? "🏆" : "⭐";
  return `${symbol} ${rating}`;
};
console.log(makeBadge(9.5));             // "🏆 9.5"
Q3

매개변수 기본값 활용

🌱 기초

교안 04-5를 참고해 기본값이 있는 함수를 작성하세요.

할 일
  • function makeTicket(title, count = 1) 함수를 만든다 — count 기본값은 1
  • `${title} ${count}매` 형태의 문자열을 리턴한다

function makeTicket(title, count = 1) {
  return `${title} ${count}매`;

Q5

return 빠뜨린 함수 고치기

🌿 응용

console.log만 있고 return이 없어서 결과를 못 쓰는 흔한 실수를 직접 고쳐보세요.

주어진 코드/데이터
function getTotal(price, count) {
  const total = price * count;
  console.log(total);    // 화면엔 잘 나오지만...
}

const receipt = getTotal(5000, 3);
console.log(`영수증: ${receipt}원`);
할 일
  • 위 코드를 그대로 실행해 receipt가 무엇으로 나오는지 확인한다 (undefined인 이유를 한 줄로 적는다)
  • getTotal 함수에 return을 추가해서 receipt가 15000이 나오도록 고친다
  • console.log(total)은 지우지 않고 return과 함께 남겨둬도 되는 이유를 생각해본다
💡 힌트
console.log는 '보여주기'이고 return은 '함수 밖으로 값을 전달하기'입니다. 둘은 전혀 다른 일을 합니다.
✅ 정답 보기
function getTotal(price, count) {
  const total = price * count;
  console.log(total);      // 디버깅용으로 남겨둬도 무방
  return total;            // 이 줄이 없어서 undefined였음!
}

const receipt = getTotal(5000, 3);
console.log(`영수증: ${receipt}원`);   // "영수증: 15000원" ✅

Q7

가변인자로 평점 평균 구하기

🌿 응용

영화 평점을 몇 개를 넘기든 평균을 계산하는 함수를 만드세요.

할 일
  • function average() 를 매개변수 없이 선언한다
  • arguments를 for문으로 순회해 합계를 구한다
  • 합계 / arguments.length 를 리턴한다
  • average(9.8, 9.5, 9.2) 와 average(8, 9) 로 테스트한다
✅ 정답 보기
function average() {
  let total = 0;
  for (let i = 0; i < arguments.length; i++) {
    total += arguments[i];
  }
  return total / arguments.length;
}
console.log(average(9.8, 9.5, 9.2)); // 9.5
console.log(average(8, 9));          // 8.5
Q8

콜백 전달 vs 즉시 실행 구분

🌿 응용

교안 04-7의 실수 사례를 직접 재현하고 고쳐보세요.

주어진 코드/데이터
function run(callback) {
  console.log("작업 시작");
  callback();
}

function sayDone() {
  console.log("완료!");
}

run(sayDone());   // 괄호가 붙어있음!
할 일
  • 위 코드를 그대로 실행해서 출력 순서를 확인한다 (예상과 다를 것)
  • 왜 "완료!"가 먼저 출력되는지 한 줄로 설명한다
  • run(sayDone)으로 괄호를 빼서 고치고, 정상 순서(작업 시작 → 완료!)가 되는지 확인한다
💡 힌트
괄호 유무 하나로 '전달'과 '실행'이 완전히 달라집니다 — 콜백으로 넘길 때는 괄호를 붙이지 않습니다.
✅ 정답 보기
// 문제 상황: run(sayDone())
// → sayDone()이 먼저 그 자리에서 실행되어 "완료!" 출력
// → run에는 sayDone()의 리턴값(undefined)이 전달됨
// → run 안의 callback()은 undefined() 호출이라 TypeError 발생

// 고친 버전
run(sayDone);   // 괄호 없이 이름만 전달
// "작업 시작" → "완료!"  순서로 정상 출력 ✅
Q9

콜백 함수 만들기

🌳 도전

배열의 각 요소에 '원하는 처리'를 적용하는 함수를 직접 만들어 forEach의 원리를 이해합니다.

할 일
  • function each(arr, callback) 을 선언한다 — arr을 for문으로 돌며 callback(요소, 인덱스)을 호출
  • each(["도시의 밤","여름의 문"], function(title, i){ console.log(i, title); }) 로 테스트한다
  • 내장 forEach로도 같은 결과를 내보고 비교한다
✅ 정답 보기
function each(arr, callback) {
  for (let i = 0; i < arr.length; i++) {
    callback(arr[i], i);
  }
}

each(["도시의 밤", "여름의 문"], function (title, i) {
  console.log(i, title);
});

["도시의 밤", "여름의 문"].forEach(function (title, i) {
  console.log(i, title);   // 동일한 결과 — forEach의 정체!
});
Q10

makeCard() — 카드 HTML 문자열 함수

🎬 index.html 미션

index.html 카드 한 장의 HTML을 그대로 리턴하는 함수를 main.js에 만듭니다. 2장의 삼항연산자 평점 배지가 여기에 들어갑니다. (출력은 아직 콘솔로 — 8장에서 이 함수가 화면을 그립니다!)

주어진 코드/데이터
<!-- index.html의 카드 구조 (참고) -->
<article class="card rounded-lg overflow-hidden flex-2" data-rating="9.2">
  <img class="w-100 h-300px object-cover" src="..." alt="...">
  <div class="card-body bg-white px-3 py-5 fs-lg">
    <h3>스텔라 오디세이</h3>
    <p class="genre text-gray">SF · 액션</p>
    <span class="badge">NEW</span>
  </div>
</article>
할 일
  • function makeCard(title, genre, rating, img) 를 선언한다
  • rating이 있으면 ⭐ 평점, 없으면 빈 문자열이 되는 badge 변수를 삼항연산자로 만든다
  • 위 카드 구조의 HTML 문자열을 템플릿 리터럴로 만들어 리턴한다 (badge는 article 여는 태그 바로 다음에)
  • makeCard("스텔라 오디세이", "SF · 액션", 9.2, "poster.jpg") 결과와, rating에 null을 넣은 결과를 콘솔로 비교한다
💡 힌트
함수에 값을 넣으면 HTML이 나온다 — 이 감각이 8장 동적 렌더링의 전부입니다.
✅ 정답 보기
// main.js (이어서 작성)
function makeCard(title, genre, rating, img) {
  const badge = rating
    ? `<span class="rating-badge">⭐ ${rating}</span>`
    : "";
  return `
    <article class="card rounded-lg overflow-hidden flex-2" data-rating="${rating}">
      ${badge}
      <img class="w-100 h-300px object-cover" src="${img}" alt="${title} 포스터">
      <div class="card-body bg-white px-3 py-5 fs-lg">
        <h3>${title}</h3>
        <p class="genre text-gray">${genre}</p>
      </div>
    </article>`;
}

console.log(makeCard("스텔라 오디세이", "SF · 액션", 9.2, "poster.jpg"));
console.log(makeCard("은하수 특급", "SF", null, "poster2.jpg")); // 배지 없음
